# Interview Prep Assistant

An AI-powered interview preparation tool built using Python and Google Gemini API.

## Features
- Works for any tech role
- Generates interview-ready answers
- Provides examples and key talking points
- Uses global industry standards

## Tech Stack
- Python
- Google Gemini API

## How to Run
1. Install Python 3.10+
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python interview_assistant.py
